# Flutter Checkout Demo Using Rapyd OpenAPI

Launch the application on Chrome using 

```bash
flutter run -d chrome --web-port {{YOUR_PORT_NUMBER}} --web-hostname 0.0.0.0 --web-browser-flag "--disable-web-security"
```
